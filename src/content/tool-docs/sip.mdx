#### What This SIP Calculator Covers

This calculator estimates the growth of your monthly SIP based on expected annual returns and supports practical assumptions often used in long-term planning:

- Monthly compounding
- Optional annual SIP step-up
- Optional initial lump-sum amount
- Inflation-adjusted final value
- Year-by-year projection table

##### Calculation Model

For each month:

1. Existing corpus grows by one month of expected return.
2. Monthly SIP contribution is added.
3. At the start of every new year, SIP amount can increase by the configured step-up percentage.

The final estimate includes:

- Total invested amount
- Estimated corpus value
- Estimated gains
- Real (inflation-adjusted) value

##### Privacy

All calculations run locally in your browser. No financial data is transmitted anywhere.
