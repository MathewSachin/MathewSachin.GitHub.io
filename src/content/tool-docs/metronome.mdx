#### How It Works

This tool runs a browser-based metronome using the
[Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API).
You can set tempo directly, adjust the time signature, or tap a button to estimate BPM from your rhythm.
Everything runs locally in your browser.

##### Features

- **Adjustable tempo:** Set BPM from 20 to 300 with both slider and number input.
- **Time signatures:** Choose beats per bar and beat unit (2, 4, 8, 16).
- **Tap tempo:** Repeated taps estimate BPM from average tap intervals.
- **Accented downbeat:** The first beat of each bar is played with a higher-pitched click.

##### Privacy

No audio data or interaction data is sent to a server.
