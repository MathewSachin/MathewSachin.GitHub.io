<h4>How It Works</h4>
<p>
  This tool converts between human-readable date/time values and
  <a href="https://en.wikipedia.org/wiki/Unix_time" target="_blank" rel="noopener">Unix timestamps</a>
  — entirely inside your browser. <strong>No data is sent to a server.</strong>
</p>

<h5 class="mt-4">What is a Unix Timestamp?</h5>
<p>
  A Unix timestamp (also called an epoch timestamp) is the number of seconds — or milliseconds
  — that have elapsed since <strong>00:00:00 UTC on 1 January 1970</strong>, known as the
  Unix epoch. It is a timezone-independent, language-agnostic way to represent a point in time
  and is widely used in databases, APIs, log files, and JWT tokens.
</p>

<h5 class="mt-4">Implementation Details</h5>
<p>
  All date arithmetic uses JavaScript's built-in
  <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date" target="_blank" rel="noopener"><code>Date</code></a>
  object — no third-party library is needed.
</p>
<ul>
  <li>
    <strong>Date/Time → Epoch:</strong>
    The <code>&lt;input type="datetime-local"&gt;</code> value is parsed with <code>new Date(val)</code>.
    <code>Date.getTime()</code> returns the Unix timestamp in <strong>milliseconds</strong>;
    dividing by 1000 and flooring gives seconds.
    The input is pre-filled with the current local time for convenience.
  </li>
  <li>
    <strong>Epoch → Date/Time:</strong>
    The raw number is checked against a threshold of 10<sup>10</sup> (approximately
    November 2286 in seconds). Values above this threshold are treated as
    <strong>milliseconds</strong>; values below are multiplied by 1000 and treated as
    <strong>seconds</strong>. The resulting <code>Date</code> object is then formatted as
    UTC, local, and ISO 8601 strings using the browser's native formatting methods.
  </li>
  <li>
    <strong>Live clock:</strong>
    A <code>setInterval</code> callback fires every second, calling <code>new Date()</code>
    and updating the current seconds, milliseconds, and UTC string in real time.
  </li>
</ul>

<h5 class="mt-4">Privacy</h5>
<p>
  All conversions run locally in your browser tab. No timestamps or dates are transmitted
  anywhere.
</p>
