How It Works

This tool tests a regular expression against sample text and draws its logic as a railroad diagram — entirely inside your browser. No data is sent to a server.

What is a Railroad Diagram?

A railroad diagram (also called a syntax diagram) is a visual representation of a formal grammar. Each token in the expression is drawn as a box, and the arrows between boxes show how text flows through the pattern — like trains on tracks. It makes alternations (choices), quantifiers (loops), and groups immediately obvious at a glance, which is why they are the standard way to document grammars in programming language specifications.

Reading the Diagram

Dashed box Capturing or non-capturing group — bundles part of the pattern

Implementation Details

The diagram is generated by a hand-written recursive-descent parser that converts the regex string into an abstract syntax tree (AST), which is then walked to produce an SVG railroad diagram. No third-party libraries are used — everything runs in the browser.

Match highlighting uses String.prototype.matchAll() with the g flag to find every match, then builds an HTML string that wraps each matched span in a <mark> element.

Privacy

All processing happens locally in your browser tab. Your patterns and test strings never leave your device.
