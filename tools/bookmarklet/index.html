What Is a Bookmarklet?

A bookmarklet is a browser bookmark whose URL begins with javascript: instead of http://. When clicked, the browser executes the embedded JavaScript on the current page — no browser extension or installation required. Common uses include toggling dark mode, extracting data from tables, tweaking page styles, or automating repetitive tasks.

The Compilation Pipeline

Your code passes through a strict, sequential pipeline:

  1. IIFE Wrapper — The code is wrapped in (function(){ … })() so local variables don't leak into the target page's global scope.
  2. Comment Stripping — Single-line (//) and block (/* … */) comments are removed. URLs such as https:// inside string literals are preserved intact.
  3. Whitespace Compression — Newlines and tabs are replaced with a single space, and runs of multiple spaces are collapsed to one — but only outside string literals.
  4. Template Literal & Regex Awareness — Content inside backtick template literals (including embedded newlines) is preserved verbatim. A / character is correctly identified as a regex delimiter or division operator based on context.
  5. URI Encoding — Characters that would break a URL bar (%, ", ', #, &, <, >, `) are percent-encoded so the browser doesn't mangle your script before it executes.
Privacy

All compilation runs entirely in your browser — your code never leaves your device.
